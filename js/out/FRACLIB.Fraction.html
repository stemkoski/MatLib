Fraction

FRACLIB. Fraction

A Fraction object represents a rational number. It has an integer numerator and (nonzero) integer denominator. Supported mathematical operations: addition, subtraction, multiplication, division, negation, inversion. Fractions are always reduced to lowest terms.

reduce() → {Fraction}

Description:
  • Reduces this fraction to lowest terms n / d, where n and d have no common factors. If the denominator is negative, multiplies numerator and denominator by -1. This method is automatically called after any change in values so that fractions are always in lowest terms.
Source:
  • matlib.js, line 132
Example
var x = new FRACLIB.Fraction(2, 10);  // x stores the fraction 2/10
x.reduce();                           // x stores the fraction 1/5
var y = new FRACLIB.Fraction(1, 9);
var z = new FRACLIB.Fraction(2, 9);
y.add( z );                           // y now stores the fraction 1/3 (automatically reduced from 3/9)
